# Todo Widget

A minimal desktop to-do widget for Windows built with Python and PySide6.

## Features

- Drag anywhere on desktop
- Resize support
- Double-click lock
- Startup fade animation
- Completed task section
- Right-click to delete tasks
- Auto-start option

## Download

Download the latest version from the Releases section.

## Built With

- Python
- PySide6
- PyInstaller
